```vim
硬件： ROG MAXIMUS XII HERO(WI-FI) / NVIDIA GTX 2080TI / DDR4 64G 3600 / SSD 2T / HDD 4T

软件： CUDA9.0-CUDA10.2 / cuDNN / TensorRT / OpenCV / caffe / YOLO-V3 / Protobuf / Matlab / VIM       
```
